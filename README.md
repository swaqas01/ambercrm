# Amber Lead Desk — Amber Homes Real Estate CRM
Private repository. Deployed via Vercel (auto-deploy on push to main).

- `src/App.jsx` — frontend application
- `api/ai.js` — serverless AI proxy (requires `ANTHROPIC_API_KEY` env var in Vercel)
- `DEPLOY-GUIDE.md` — step-by-step hosting guide

**Roadmap:** v1 foundation (this) → Supabase backend (real auth, leads DB, RLS) → security hardening → reports & commissions.
